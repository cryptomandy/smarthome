* import 和 require区别
commonjs 的 require 是加载时执行; 而 es6 的 import 是调用时执行，加载时只产生引用
比如引入模块里的值异步的产生了变化， require 他的文件的值是不变的; 而 import 的，则会根据模块值的变化而变化
